# Paradigma Components: DanceScore (AX09)

## Architettura
I componenti devono essere **Serious, Professional, Elegant (AX00C)**. 

## Assiomi Applicati
- **AX00C**: Eredità rigorosa di stili HSL da `index.css`. No Arlecchino.
- **AX06**: I form devono essere precompilati (Context-First). No "Empty Forms".
- **AX04**: I componenti di dashboard devono mostrare stati simultanei.

## Rischi
L'aggiunta di stili "vibrant" o animazioni ludiche compromette l'autorità del sistema gestionale di gara.
